# RG Dashboard Ejecutivo - RGrlk Group

## Descripción
Sistema estratégico de control y radar de oportunidades de carrera para Rodrigo Garelik. 
Este repositorio centraliza la inteligencia competitiva y el scoring de posiciones C-Level y Consultoría.

## Arquitectura
- **Modelo:** SPA (Single Page Application) - Vanilla JS.
- **Base de Datos:** CSV (`/data/dashboard_oportunidades_rgrlk.csv`).
- **Seguridad:** Almacenamiento Local (Privacy Absolute).
